# react-template

## What's included?

- Vite
- React (TypeScript)
- Tailwind CSS
- React Router DOM
- Prettier

## Getting Started
You can click `Use this template` to create a new repo or you can clone it.

```bash
git clone https://github.com/Mahcks/react-template.git react-template
cd react-template
npm install
npm run dev
```
